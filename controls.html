Controls & Shortcuts

Mouse buttons, keyboard shortcuts, and menubar reference for both modes.

Mouse Controls

Note: Requires Logi Options+ to remap MX Anywhere 3S side buttons to keyboard shortcuts. The side buttons don't generate standard macOS CGEvents — Hammerspoon cannot detect them directly.
Button Remapped To Action Mode
Forward (top side button) Ctrl+Shift+Cmd+9 Start streaming — overlay appears, words flow in real-time Streaming
Back (bottom side button) Ctrl+Shift+Cmd+0 Stop streaming — text copied to clipboard and pasted Streaming
Middle (below scroll wheel) Ctrl+Shift+Cmd+8 Re-transcribe last recording — grabs latest WAV from archive, re-processes with medium.en model, pastes result Both

Keyboard Shortcuts

Ctrl+Cmd+Q Stop recording → transcribe → auto-paste Batch

Menubar

The menubar icon displays the current system state:

  • ○ = idle, ready
  • ● 0:05 = batch recording in progress (with running timer)
  • ◉ 0:05 = streaming in progress (with running timer)

Menubar menu items:

  • Toggle Recording / Stop Recording — starts or stops batch mode
  • Re-transcribe Last — re-processes the last audio recording
  • ◉ Start Streaming / ⏹ Stop Streaming — streaming mode controls
  • Model selection submenu — choose between available whisper models
  • Post-processing mode — toggle correction features
  • Notifications toggle — enable/disable macOS notifications
  • Sounds toggle — enable/disable sound effects

Logi Options+ Setup

  1. Download and install Logi Options+ from logitech.com
  2. Open Logi Options+ → your MX Anywhere 3S should appear
  3. Click on the mouse image to see button assignments
  4. Click on the Forward button (top side button)
  5. Choose "Keyboard shortcut" as the action type
  6. Press Ctrl+Shift+Cmd+9 in the shortcut field
  7. Save
  8. Click on the Back button (bottom side button)
  9. Choose "Keyboard shortcut" as the action type
  10. Press Ctrl+Shift+Cmd+0 in the shortcut field
  11. Save
  12. Test: press Forward button — Hammerspoon menubar should change to ◉ and overlay should appear
  13. Click on the Middle button (below the scroll wheel)
  14. Choose "Keyboard shortcut" as the action type
  15. Press Ctrl+Shift+Cmd+8 in the shortcut field
  16. Save
  17. Test: record something with Forward/Back, then press Middle button — it should re-transcribe and paste the result
Warning: Without Logi Options+, the mouse side buttons simply won't work for streaming. You can still use the keyboard shortcuts Ctrl+Shift+Cmd+9 and Ctrl+Shift+Cmd+0 as alternatives.

User Experience — Batch Mode

  1. You're typing in any app (VS Code, browser, terminal, email...)
  2. Press Ctrl+Cmd+W — menubar changes from ○ to ● 0:00
  3. Speak your text naturally — timer counts up in menubar
  4. Press Ctrl+Cmd+Q — menubar shows processing indicator
  5. After 2-4 seconds, transcribed text appears at your cursor position
  6. Menubar returns to ○

User Experience — Streaming Mode

  1. You're in any app where you want to type by voice
  2. Press mouse Forward button — dark overlay appears at screen bottom, red ● STREAMING badge
  3. Speak — words appear on overlay in real-time (gray = partial, white = confirmed)
  4. Completed sentences get auto-corrected by Copilot (green flash)
  5. Continue speaking as long as you want — text accumulates
  6. Press mouse Back button — overlay disappears, all text is pasted at cursor
  7. Audio is automatically saved to recordings/ as a WAV file
  8. Menubar returns to ○

Re-transcribe

If the streaming result wasn't accurate enough, press the Middle mouse button (or Ctrl+Shift+Cmd+R) to re-transcribe the last recording. This uses the larger medium.en model (1.5GB) for higher accuracy — it takes a few seconds but usually produces better results.

Re-transcribe works for the most recent recording from either batch or streaming mode — whichever was last.
