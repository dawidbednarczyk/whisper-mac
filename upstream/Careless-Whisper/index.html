Careless Whisper

Local speech-to-text for macOS. Press a hotkey anywhere, speak, get text pasted — no cloud, no subscription.

Quick Start

git clone https://github.com/Bloodw0lf/Careless-Whisper.git ~/Scripts/Whisper
cd ~/Scripts/Whisper
./install.sh

The installer handles everything:

  1. Installs ffmpeg + whisper-cpp via Homebrew
  2. Offers to install Hammerspoon
  3. Downloads a whisper model (~800 MB)
  4. Creates config & wires up Hammerspoon
  5. Launches Hammerspoon and reloads config

After install: press Ctrl+Cmd+W to start recording, press again to transcribe and paste.

Hotkeys

Shortcut Action
Ctrl+Cmd+W Toggle recording — first press starts, second press stops + transcribes + pastes
Ctrl+Cmd+Q Emergency stop

Configurable in whisper-stt.conf. Reload Hammerspoon to apply changes.

Menubar

Symbol State
○ Idle
● 0:12 Recording (with elapsed time)
⠋⠙⠹… Transcribing

Dropdown Menu

Click the menubar icon for Toggle / Stop controls, Notifications / Sounds toggles, a model selector (switch between all .bin files in models/), and a list of recent transcriptions (click to copy).

Models

Model Size Speed Quality Best for
ggml-large-v3-turbo ~800 MB Fast High Recommended — daily use
ggml-large-v3 ~1.5 GB Slow Highest Long recordings, noisy audio
ggml-medium ~1.5 GB Med Good Balanced alternative

All models are multilingual with auto-detection. Switch via menubar — no restart needed. More models at Hugging Face.

Architecture

Ctrl+Cmd+W
   ↓
Hammerspoon (whisper_hotkeys.lua)
   ↓ hs.task.new()
whisper.sh toggle
   ├─ [start] ffmpeg records → $TMPDIR/whisper_recording.wav
   └─ [stop]
        ├─ touch $TMPDIR/whisper_transcribing  ← spinner
        ├─ whisper-cli transcribes
        ├─ rm $TMPDIR/whisper_transcribing  ← done
        ├─ append to history.txt
        ├─ copy to clipboard
        └─ auto-paste via AppleScript

Configuration (whisper-stt.conf)

Variable Default Description
WHISPER_MODEL_PATH models/ggml-large-v3-turbo.bin Active model (or use menubar)
WHISPER_LANGUAGE auto Language code or auto
WHISPER_TRANSLATE 0 1 = translate to English
WHISPER_AUTO_PASTE 1 Auto-paste after transcription
WHISPER_AUDIO_DEVICE default macOS input device or AVFoundation index
WHISPER_MAX_SECONDS 7200 Max recording duration (seconds)
WHISPER_HISTORY_MAX 10 Entries kept in history
WHISPER_NOTIFICATIONS 1 0 = disable notifications
WHISPER_SOUNDS 1 0 = disable sounds
WHISPER_HOTKEY_TOGGLE ctrl,cmd,w Toggle hotkey
WHISPER_HOTKEY_STOP ctrl,cmd,q Emergency stop hotkey

Troubleshooting

  • Hotkey does nothing — Hammerspoon config not reloaded. Menubar → Reload Config.
  • No audio captured — Wrong device. Run whisper.sh list-devices and set WHISPER_AUDIO_DEVICE.
  • whisper-cli not found — Re-run ./install.sh or brew install whisper-cpp.
  • Spinner stuck — Auto-clears after 10 min. Check $TMPDIR/whisper-error.log.
  • Hotkey ignored while busy — Expected. Wait for transcription to finish.
  • Model rejected for language — .en.bin models are English-only. Switch model or set WHISPER_LANGUAGE=en.

Manual Testing

./whisper.sh status         # current state + version
./whisper.sh list-devices   # audio inputs
./whisper.sh start          # start recording
./whisper.sh stop           # stop + transcribe

Uninstall

./uninstall.sh

Removes Hammerspoon integration and temp files. Optionally uninstalls whisper-cpp, ffmpeg and Hammerspoon via Homebrew (asks per package). The repo directory is kept.
