Whisper Speech-to-Text

System-wide speech-to-text on macOS using local Whisper AI. Record with hotkeys or mouse buttons, transcribe locally, paste anywhere.

🎤

Batch Mode Keyboard

Press Ctrl+Cmd+W to record, Ctrl+Cmd+Q to stop. Audio is transcribed by whisper-cli (medium.en model) and pasted into any app. Best for long dictation.

⚡

Live Streaming Mouse

Press mouse forward button to start. Words appear on screen in real-time via whisper-stream (base.en model). Copilot auto-corrects grammar. The star feature. Learn more →

Key Highlights

  • ✅ 100% local transcription (no cloud, privacy-first)
  • ✅ Works in ANY app — browser, terminal, IDE, email
  • ✅ Self-healing microphone detection (auto-corrects when USB devices reconnect)
  • ✅ Real-time streaming with live overlay and AI grammar correction
  • ✅ Audio archiving — all recordings (batch + streaming) saved as WAV for re-transcription
  • ✅ Apple Silicon optimized (Metal-accelerated, ~55ms per streaming step on M-series)

Quick Start

  1. Install prerequisites → Installation Guide
  2. Clone the repos and download models → Installation Guide
  3. Configure Hammerspoon and mouse buttons → Controls & Shortcuts
  4. Press Ctrl+Cmd+W to record, Ctrl+Cmd+Q to transcribe (batch mode)
  5. Press mouse Forward button for live streaming → Live Streaming
